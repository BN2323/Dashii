# Dashii 🎮📦

**Dashii** is a fast-paced, **Geometry Dash-inspired game** built using **Phaser and React**.  
Players control a box character navigating through challenging obstacles, testing their reflexes, timing, and precision.

---

## ✨ Features

- 🎯 **Realistic Physics**: The player box can land on any side, adding dynamic gameplay.  
- 🪨 **Obstacle Challenges**: Includes meteoroids and other obstacles that require timing and strategy.  
- ⏩ **Smooth Gameplay**: Fast-paced levels with responsive controls.  
- 🏆 **Score Tracking**: Keep track of high scores and progress.  
- 📱 **Browser-Based**: Playable directly in a web browser.  
- 🎨 **Interactive UI**: Clean and engaging interface for a better gaming experience.

---

## 🛠️ Built With

- **Phaser 3** – Game engine for physics and interactive gameplay  
- **React** – Frontend framework for UI and game integration  
- **JavaScript / HTML / CSS** – Core web technologies
